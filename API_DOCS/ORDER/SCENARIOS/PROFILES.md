# Scénarios de Profils Entreprises

Chaque type d'entreprise a des attentes et des contraintes logistiques différentes. L'outil universel **Cas G** s'adapte à ces profils.

## 1. Le Profil "E-commerce & Retail"
- **Attente** : Rapidité et visibilité.
- **Scénario type** : G.2 (Éclaté) ou G.3 (Milk Run).
- **Contraintes clés** : 
  - Livraisons "Aussitôt que possible".
  - Preuve de livraison (Photo/Signature).
  - Gestion des retours (G.5).

## 2. Le Profil "B2B & Distribution Industrielle"
- **Attente** : Ponctualité et volumes.
- **Scénario type** : G.1 (Conséquent) vers des entrepôts.
- **Contraintes clés** : 
  - Fenêtres horaires strictes (Time Windows).
  - Capacités de chargement critiques (Poids/Dimensions).
  - Séquençage des livraisons (Relation de précédence).

## 3. Le Profil "Services à la Personne / On-demand"
- **Attente** : Flexibilité et confiance.
- **Scénario type** : G.6 (Point de passage / Maintenance).
- **Contraintes clés** :
  - Compétences spécifiques (Skills) du driver.
  - Temps de service longs (Service Time).
  - Géolocalisation temps réel haute précision.

## 4. Le Profil "Prestataire Logistique (3PL)"
- **Attente** : Rentabilité et optimisation de flotte.
- **Scénario type** : G.4 (Cross-Docking) et G.3 (Multi-tours).
- **Contraintes clés** :
  - Optimisation Multi-Drivers (Global Dispatch).
  - Gestion des pannes et transferts de tournées.
  - Matrice de coûts personnalisée.
